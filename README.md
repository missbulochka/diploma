# diploma

## Prerequisites

### devcontainers/cli
Install the Dev Container CLI:

```bash
npm install -g @devcontainers/cli
```

## Requirements

- go 1.21.3
- docker 26.0.0
- docker-compose 1.29.2
- devcontainers/cli (optional)

## Run
You can run dockerized application via docker compose:
```bash
docker compose up --build
```
